# 📸 摄影师拍摄管理系统 - 使用说明

## 访问地址

| 功能 | 链接 |
|------|------|
| 管理后台 | https://photoshoot-manager-production-0133.up.railway.app |
| 日历订阅 | https://photoshoot-manager-production-0133.up.railway.app/calendar.ics |

## 一、添加拍摄事件

1. 手机浏览器打开管理后台
2. 点击右下角 **+** 按钮
3. 填写拍摄信息：
   - 模特姓名、拍摄日期和时间
   - 拍摄内容（如"古风人像"）、拍摄地点
   - 道具：输入名称按回车添加
   - 定金/尾款金额、是否已收到
4. 点击「添加事件」

## 二、订阅手机日历（核心功能）

**iOS：**
打开「日历」App → 底部点「日历」→「添加日历」→「添加订阅日历」→ 粘贴日历订阅链接 → 点「订阅」

**Android：**
打开「Google 日历」App → 左上角菜单 →「设置」→「添加日历」→「通过网址」→ 粘贴链接 →「添加日历」

订阅后手机会自动同步，并在拍摄前一天 + 拍摄前 3 小时弹出提醒。

## 三、道具清单

点击底部「道具」标签，自动汇总当月所有拍摄需要的道具，点击可标记已准备。

## 四、收入统计

点击底部「收入」标签，按月/按年查看已收定金、尾款和总收入。

## 五、月历视图

在「拍摄」页顶部可切换「列表/月历」视图。

## 六、暗色模式

点击右上角月亮/太阳图标切换。

## 七、注意事项

1. 日历订阅后，手机每 15-30 分钟自动刷新，添加新拍摄后稍等即可同步
2. 日历提醒完全依赖手机系统，无需保持浏览器打开
3. 道具准备状态保存在手机浏览器中，不同设备之间不共享
